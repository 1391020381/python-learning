# windows和linux的目录结构
  * windows有分盘,linux没有,linux直接对应文件夹
  * bin
  * boot与开机有关
  * etc配置有关
  * dev和设备有关
  * lib与库有关
  * home<家目录,登录后的路径>
  * /home/ubuntu<根目录下的home文件夹下的ubuntu文件夹>
  * touch
  * mkdir
# 命令使用方法
  * Linux命令格式
     * command [-options] [parameter]  
     * ls /home(在对应文件夹下的东西)
     * ls -[选项]
     * 以.开头的文件是隐藏文件
     * ls -a (显示所有文件)
     * ls -l (以列表的格式显示)
     * ls -h -l -a等同于<ls -lah>
